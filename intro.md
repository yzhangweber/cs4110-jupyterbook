# Decode Computation︰ Hands-On Formal Languages and Algorithms

Welcome to **Decode Computation**, a free, interactive textbook for **CS 4110: Formal Languages and Algorithms**. This book makes the abstract world of computation **tangible, hands-on, and fun**.

## Accessibility Settings for Screen Reader Users

If you use a screen reader to navigate this textbook, the following settings are recommended for the best experience.

### Reading Code Examples

Python code examples in this book contain characters such as parentheses `()`, brackets `[]`, colons `:`, and underscores `_` that are essential to understanding the syntax. By default, most screen readers skip or abbreviate punctuation. To hear all code characters correctly, enable the **Read All Punctuation** setting in your screen reader:

- **VoiceOver (Mac/iOS):** `VO+V` → Verbosity → Punctuation → select **All**
- **NVDA (Windows):** Press `NVDA+P` repeatedly to cycle through punctuation levels until **All** is announced
- **JAWS (Windows):** Open Settings Center → Text Processing → Punctuation Level → select **All**

You can switch back to your normal punctuation level when reading non-code sections of the book.

### Reading Mathematical Notation

All mathematical notation in this book is accessible to screen readers without any additional settings. Math expressions will be announced automatically as you navigate through the text.
